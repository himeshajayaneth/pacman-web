# Contributor Code of Conduct

We want this project to be a welcoming community for everyone.

Our standards

- Be respectful and considerate.
- Assume good intent and communicate clearly.
- Report unacceptable behavior to the repository owner by opening an issue.

Consequences

Unacceptable behavior will result in removal of comments or blocking from contributing, depending on severity.

Contact

If you experience or witness unacceptable behavior, please open an issue or contact the repo owner.
