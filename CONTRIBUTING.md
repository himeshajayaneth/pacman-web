# Contributing to Pac-Man Web

Thanks for your interest in contributing! We welcome fixes, enhancements, and documentation improvements.

How to contribute

1. Fork the repository and create a feature branch: `git checkout -b my-fix`.
2. Make your changes and include clear commit messages.
3. Test locally by opening `index.html` in a browser and ensuring the game runs.
4. Push to your fork and open a Pull Request describing the change.

Reporting bugs

- Open an issue with steps to reproduce, expected vs actual behavior, and screenshots if available.

Coding style

- This is a small vanilla JS project. Keep changes minimal and self-contained.
